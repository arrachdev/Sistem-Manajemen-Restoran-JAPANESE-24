# Sistem Manajemen Restoran JAPANESE 24

Program Java berbasis objek yang dibuat untuk memenuhi Tugas Praktik 3 mata kuliah Pemrograman Berbasis Desktop.

## Fitur

- Menampilkan menu restoran
- Menambahkan menu baru
- Mencatat pesanan pelanggan
- Menghitung total pembayaran
- Menerapkan diskon
- Menampilkan struk pesanan
- Menyimpan dan memuat data menu dari file
- Menyimpan struk pesanan ke file

## Konsep OOP yang Digunakan

- Abstraksi
- Inheritance
- Encapsulation
- Polymorphism
- Exception Handling
- ArrayList
- File I/O

## Struktur Program

- MenuItem.java
- Makanan.java
- Minuman.java
- Diskon.java
- Menu.java
- Pesanan.java
- RestaurantApp.java
